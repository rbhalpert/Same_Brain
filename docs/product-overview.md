# Same Brain — Product Overview

## What it is
Same Brain is a browser-first phone party game where players try to think like the rest of the group.

Each round, all players receive the same prompt and privately submit a short answer on their phones. After answers are locked, the game reveals them, groups obvious matches, and awards points based on how many players gave the same answer.

## Core product thesis
The fun comes from social intuition:
- “I was going to say that.”
- “You and I think the same way.”
- “I overthought it.”
- “Why did three of us say goose?”

This is not a trivia game, drawing game, or bluffing game.
This is a social matching game.

## Primary goals
- extremely low friction
- join instantly by QR, link, or room code
- no account required
- phone-first play
- shared screen strongly supported
- fast rounds
- satisfying reveal
- easy to understand after one round

## MVP
Build a fully playable vertical slice:
- host creates room
- players join
- lobby works
- 8-round Classic mode
- prompt -> answer -> reveal -> score -> leaderboard loop
- final results
- rematch

## What must not be lost
- simplicity
- fast pacing
- clean phone UI
- reveal as the emotional payoff
- trust in scoring and grouping
- broad social accessibility

## Out of scope for MVP
- accounts
- progression
- purchases
- chat
- voice
- advanced semantic AI grouping
- extra modes beyond stub-ready structure
